# MarionetteJS_Simple_App:

	This simple application based on MarionetteJS with Gulp & localStorage

# Init project:

	1. npm install
	2. bower install
	3. gulp watch

# Technologies list:

Frameworks:

	CSS:
	1. Normalize.CSS
	2. Twitter Bootstrap 3
	3. AdminLTE 2
	
	JS:
	1. jQuery
	2. Underscore
	3. Backbone
	4. Backbone.localStorage
	5. MarionetteJS
	
Technologies:

	1. NPM
	2. Bower
	3. Gulp
	4. LESS
